プログラム名：DuelBot

某デュエルシミュレーターの身内向け支援ツール(?)です。
DiscordのBotとして動作する形式になっています。

紙でやっていたように皆のデッキをぶちこんでごちゃまぜにしてランダムなデッキでやりたい！とか登録しておいてあいつのデッキを参考に、といったものができないかと頼まれてやりました。

導入方法：使用したいサーバーのトークンを取得してDuelBot.pyの上の方にあるトークン欄にペーストした上で同pyを実行。
実際にはHerokuで24時間稼働させてました。


できる事

・ADSにおけるパスワード形式でのデッキの登録・上書き・削除
・↑で登録されたデッキのランダム・指定抽出
・VCにデュエル用のBGMを流す(予めフォルダにセットしておく必要アリ)
・Discordの書き込み・VC接続のログ取り(管理用)


実装予定
・デッキと同じく音楽もbotを通して追加可能にする。

つかいかた
DiscordのBotらしく/と単語を読み取って動作します。


[Duel bot 操作コマンド　"/" +下記の単語で発動]
"show_deck"格納済みデッキ一覧を返す
"deck_add デッキ名 内容(ydke:~)"デッキ追加(同作者同名デッキは上書き)
"deck_del デッキ番号"指定の番号のデッキを消す
"deck_num デッキ番号"指定番号のデッキの中身の表示

"duel"ランダムタッグ(2vs2)配布(パスワード制)
"duel_s"ランダム(1vs1)配布(パスワード制)
"reset"↑の配布におけるデッキ選定のリセット
"bgm"bgmを流す
-----------------BGM再生中に使用可能-------------------
"next"bgmを変える
"change""bgmの再生リストを変える
"volu""ボリュームアップ
"vold""ボリュームダウン
"stop"botをボイスチャンネルから切断する
"show_bgm""bgmのタイトルを教えてくれる
"plz_bgm"bgmの音声ファイルをくれる※テスト中
"----------------------------------------------------------------------
"del"Botによる発言とBotに対するコマンド宣言の削除
